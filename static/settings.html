クライアント ID
Azure Portal で登録したアプリケーション (クライアント) ID を入力してください。空欄の場合はデフォルト値が使用されます。
独自クライアント ID の取得方法
  1. portal.azure.com でアプリを登録
  2. サポートされるアカウントの種類: 「任意の組織ディレクトリ内のアカウント」
  3. リダイレクト URI: 「パブリッククライアント」→ http://localhost
  4. API のアクセス許可: Microsoft Graph → Mail.Read
  5. 認証 → 「パブリッククライアントフローを許可する」を「はい」に

Google Cloud Console での設定手順
  1. console.cloud.google.com で新しいプロジェクトを作成
  2. 「API とサービス」 → 「ライブラリ」 → Google Calendar API を有効化
  3. 「認証情報を作成」 → 「OAuth クライアント ID」 → 「ウェブ アプリケーション」
  4. リダイレクト URI に http://127.0.0.1 を追加
  5. 「OAuth 同意画面」でテストユーザーに自分のメールアドレスを追加
