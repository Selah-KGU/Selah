# 利用規約

最終更新日: 2026 年 4 月 17 日

本利用規約（以下「本規約」）は、Selah（以下「本アプリ」）の利用条件を定めるものです。本アプリをダウンロードまたは使用することにより、本規約に同意したものとみなされます。

## 本アプリについて

本アプリは、関西学院大学の教務システム等にアクセスするための**非公式**デスクトップクライアントです。

- 本アプリは関西学院大学との提携・承認は一切ありません
- 本アプリは個人が開発するオープンソースプロジェクトです
- 本アプリは [MIT License](https://github.com/mirai-mamori/Selah/blob/main/LICENSE) の下で提供されます

## 利用条件

### アカウント

本アプリを利用するには、関西学院大学の有効な SSO アカウントが必要です。本アプリはユーザー自身のアカウントでのみ使用してください。

### 許可される使用

- 自身の授業情報・課題・成績・メールの閲覧
- カレンダーへの時間割エクスポート
- AI 機能を使った学習支援（自身の API キー使用）

### 禁止される使用

- 他者のアカウントへの不正アクセス
- 大学システムへの過度な負荷をかける行為
- 本アプリを利用した自動化されたスクレイピング・データ収集
- 取得したデータの第三者への無断提供

## 免責事項

### 保証の否認

本アプリは**現状有姿 (AS IS)** で提供されます。開発者は以下について一切の保証を行いません：

- 本アプリの動作が中断しないこと、またはエラーがないこと
- 表示されるデータの正確性・完全性・最新性
- 本アプリが特定の目的に適合すること
- 大学システムの仕様変更に伴う機能の継続性

### 責任の制限

開発者は、本アプリの使用または使用不能に起因する以下を含むいかなる損害についても責任を負いません：

- 課題の提出漏れ・提出ミス
- 成績情報・時間割情報の表示の誤り
- 履修登録に関連する問題
- データの損失
- その他直接的・間接的な損害

::: warning 重要
成績・時間割・課題締切などの重要な情報は、必ず大学の公式システム（KWIC・Luna）でも確認してください。本アプリの表示のみに依存しないでください。
:::

### 課題の提出

本アプリから Luna への課題提出機能を利用する場合、提出の成否は大学のサーバーの状態に依存します。重要な提出物については、Luna の公式 Web サイトからも提出状況を確認してください。

## AI 機能に関する免責

AI 機能（履修分析・学習計画・通知サマリー）は、OpenAI / Google Gemini の API を利用して生成されたものであり、以下の点にご注意ください：

- AI の出力は参考情報であり、正確性は保証されません
- 履修に関する最終的な判断はご自身の責任で行ってください
- AI 機能の利用に伴う API 利用料はユーザーの負担となります

## 大学システムとの関係

- 大学のシステム仕様が変更された場合、本アプリの一部または全部の機能が使用できなくなる可能性があります
- 大学による KWIC・Luna 等のメンテナンス期間中は本アプリも使用できません
- 大学が本アプリからのアクセスを制限する場合、開発者はこれに従います

## 規約の変更

本規約は予告なく変更される場合があります。変更後も本アプリを継続して使用する場合、変更後の規約に同意したものとみなされます。
