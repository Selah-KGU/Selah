# TODO / 課題管理

Luna (LMS) の課題・提出物を一元管理します。

## 課題一覧

Luna に登録されている課題をリスト形式で表示します。

- 未提出 / 提出済のステータス管理
- 締切日の表示
- 期限超過の課題は赤色で強調表示

## 課題の詳細

課題をクリックすると、別ウィンドウで詳細を確認できます。

- 課題の説明文
- 添付ファイル
- 提出状況

## レポート提出

詳細ウィンドウから直接レポートを提出できます。

1. 課題の詳細を開く
2. ファイルを添付（ドラッグ&ドロップまたはファイル選択）
3. コメントを入力（任意）
4. 「提出」ボタンをクリック

::: warning
提出は Luna のサーバーに直接送信されます。提出前に内容を十分確認してください。
:::

## 掲示板投稿

Luna の掲示板 (Forum) スレッドへの返信投稿にも対応しています。

## AI 学習アシスト

OpenAI / Gemini API またはローカル AI を使い、時間割・シラバス・締切を踏まえた学習計画を生成します。

- タスクごとの優先度と取り組み方ガイド
- 結果は 6 時間キャッシュされます

::: info
AI 学習アシストを使用するには、設定画面で AI プロバイダ（ローカルモデルまたは API キー）を登録する必要があります。
:::
