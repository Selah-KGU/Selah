# ホーム

ダッシュボード画面。ログイン後に最初に表示される画面です。

## 挨拶メッセージ

時間帯に応じた挨拶メッセージが表示されます。

| 時間帯 | メッセージ |
|---|---|
| 朝（5:00 - 11:59） | おはようございます |
| 昼（12:00 - 16:59） | こんにちは |
| 夕（17:00 - 20:59） | こんばんは |
| 夜（21:00 - 4:59） | おつかれさまです |

## NOW / NEXT カード

現在進行中の授業、または次に始まる授業をカード形式で表示します。授業名・教室・時限が一目で分かります。

## お知らせ

KWIC と Luna のお知らせを統合し、最新 3 件をダッシュボードに表示します。クリックで詳細を確認できます。

## 今日のスケジュール

今日・明日の授業一覧をタイル形式で表示。時限順に並んでいるので、一日の予定をすぐに把握できます。

## 締切間近の課題

提出期限が 5 日以内の課題を緊急度別に色分けして表示します。

| 色 | 残り日数 |
|---|---|
| 赤 | 1 日以内 |
| オレンジ | 2 - 3 日 |
| 黄 | 4 - 5 日 |

## 天気情報

Open-Meteo API を使い、西宮上ケ原キャンパスの気温・天候を表示します。通学の参考にどうぞ。

- 現在の気温・天候
- 翌日の最高・最低気温と天候予報

::: info
位置情報は取得しません。西宮上ケ原キャンパスの固定座標のみを使用します。
:::

## ポータルリンク

KWIC ポータルのメインリンクをカード形式で表示します。各カードをクリックするとサブポータル画面に移動し、関連リンクやお知らせを確認できます。ICT ツールカテゴリは重複を避けるため自動的に非表示となります。
