# 時間割

KWIC と Luna のデータを統合した週間時間割ビューです。

## グリッド表示

月曜日から土曜日、1 限から 7 限までのグリッド形式で時間割を表示します。

- KWIC に登録された正規の授業
- Luna のコース情報
- シラバスお気に入りのオーバーレイ表示

各セルをクリックすると、授業の詳細情報を確認できます。

## 週単位ナビゲーション

前週 / 次週ボタンで週を切り替えられます。休講・補講・教室変更がある場合は、該当するセルにマークが表示されます。

## 試験時間割

試験期間中は、通常の時間割に加えて試験時間割の表示に切り替えることができます。

## Google カレンダー同期

Google Calendar API を使用して時間割を外部カレンダーへ自動エクスポートできます。

- Google アカウントとの OAuth 連携が必要です（アプリ内ブラウザから認可）
- 変更検知による差分同期（追加・変更・削除を自動反映）
- 設定画面から同期間隔の指定や自動同期のオン / オフが可能

## AI 履修分析

ローカル AI または OpenAI / Gemini を使い、成績・履修状況・シラバスを基に履修アドバイスレポートを生成します。

- 卒業要件に対する進捗の可視化
- おすすめ科目の提案
- 履修バランスの分析

::: info
AI 履修分析を使用するには、設定画面で AI プロバイダ（ローカルモデルまたは API キー）を構成する必要があります。
:::
