# 時間割

KWIC と Luna のデータを統合した週間時間割ビューです。

## グリッド表示

月曜日から土曜日、1 限から 7 限までのグリッド形式で時間割を表示します。

- KWIC に登録された正規の授業
- Luna のコース情報
- シラバスお気に入りのオーバーレイ表示

各セルをクリックすると、授業の詳細情報を確認できます。

## 週単位ナビゲーション

前週 / 次週ボタンで週を切り替えられます。休講・補講・教室変更がある場合は、該当するセルにマークが表示されます。

## 試験時間割

試験期間中は、通常の時間割に加えて試験時間割の表示に切り替えることができます。

## カレンダー同期

時間割を外部カレンダーに自動エクスポートできます。

### Apple カレンダー（macOS）

macOS ネイティブの EventKit を使用して、Apple カレンダーに直接イベントを作成します。

- 変更検知による差分同期（追加・変更・削除を自動反映）
- 初回同期時にカレンダーアクセスの権限許可が必要です

### Google カレンダー

Google Calendar API を使用して同期します。

- Google アカウントとの OAuth 連携が必要です
- 差分同期に対応

## AI 履修分析

OpenAI / Gemini API を使い、成績・履修状況・シラバスを基に履修アドバイスレポートを生成します。

- 卒業要件に対する進捗の可視化
- おすすめ科目の提案
- 履修バランスの分析

::: info
AI 履修分析を使用するには、設定画面で API キーを登録する必要があります。
:::
