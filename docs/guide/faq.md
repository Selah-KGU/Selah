# FAQ

## 全般

### Selah は大学公式のアプリですか？

いいえ。Selah は個人が開発したサードパーティのデスクトップクライアントであり、大学の公式アプリケーションではありません。

### 無料で使えますか？

はい。Selah は source-available な非商用ライセンスで公開されており、個人利用・教育利用などの非商用目的では無料で使用できます。商用利用には別途許可が必要です。AI 機能を使う場合は、別途 OpenAI / Google の API 利用料がかかります。

### ログイン情報はどこに保存されますか？

ログイン時の認証情報は macOS の Keychain、Windows の Credential Store に保存されます。アプリのコード内にパスワードが平文で保存されることはありません。

## トラブルシューティング

### ログインできません

- 利用者 ID とパスワードが正しいか確認してください
- 大学の SSO サービスが一時的にダウンしている可能性があります。しばらく時間をおいて再試行してください
- VPN 経由の場合、接続が不安定な場合があります

### データが更新されません

- インターネット接続を確認してください
- アプリを再起動してみてください
- セッションが切れている場合は、再ログインが必要です

### AI 機能が動作しません

- 設定画面で API キーが正しく入力されているか確認してください
- API キーの利用制限に達していないか、OpenAI / Google のダッシュボードで確認してください

### macOS で「開発元が確認できない」と表示されます

**システム設定 > プライバシーとセキュリティ** から「このまま開く」を選択してください。
